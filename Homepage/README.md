# 🌐 CS50 Homepage Project

This project was developed as part of Harvard's CS50 course. It consists of a personal multi-page website built using HTML, CSS, JavaScript, and Bootstrap.

The goal of this project is to demonstrate the ability to design and structure a responsive website, implement navigation across multiple pages, and apply basic interactivity.

---

## 👤 About the Project

This homepage represents my personal portfolio and professional transition into the field of Data Analytics.

I come from an engineering background with over 15 years of experience working on residential and commercial systems. Throughout my career, I developed strong analytical thinking and problem-solving skills.

I am currently transitioning into the data field and will begin a Higher Diploma in Science Computing at the National College of Ireland in September.
