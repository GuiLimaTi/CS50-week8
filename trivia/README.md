# 🎯 CS50 Trivia Web App

An interactive trivia web application developed as part of Harvard’s CS50 course.  
This project demonstrates core front-end development skills, including DOM manipulation, event handling, and dynamic user feedback using JavaScript.

---

## 🚀 Features

- Multiple choice question with real-time feedback  
- Free-response input with answer validation  
- Dynamic color feedback (green for correct, red for incorrect)  
- Clean and user-friendly interface  

---

## 🧠 What I Learned

- DOM manipulation using querySelector and querySelectorAll  
- Event handling with addEventListener  
- Conditional logic for validating user input  
- Structuring a simple front-end application  
- Creating interactive experiences with JavaScript  

---

## 🛠️ Technologies Used

- HTML5

- JavaScript

## 🎮 How It Works

Users answer a multiple-choice question by clicking a button.  
The app provides immediate feedback by changing colors and displaying messages.  

Users can also type an answer in a free-response input field.  
The system validates the answer and gives instant feedback.
